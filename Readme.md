#ColoClock
基于WPF的简易时钟程序

---------------------------------------
##简介
基于WPF + C# 的时钟程序，实现了多个计时器和闹钟并发的功能

---------------------------------------
##版本更新

---------------------------------------
###19/5/21 1.0.0.1?

*  新增了  .gitignore 和 Readme.md 文件

###19/5/16 1.0.0.0
*  项目上传至GitHub上

*  基础的时间显示 屏幕中央显示

*  基础时钟功能 到达时间自动播放音乐 并在Win10系统下会有系统通知弹窗

*  基础定时器功能 到达预定时间 Win10系统下会有系统通知弹窗

*  添加设定与修改窗口 双击“+”按钮 或者双击当前项目 可以添加/修改项目

*  点击左下角的小风扇可以换壁纸呦

*  窗口的缩放与空间的自适应
